# 🚀 Deploy Your Website to GitHub Pages

## Your Repository
- **Repository Name:** `my-photography-sites`
- **Repository URL:** `https://github.com/CHAITHANY-123/my-photography-sites.git`

---

## Method 1: Using GitHub Website (Easiest - No Command Line)

### Step 1: Upload Your Files

1. Go to your repository: `https://github.com/CHAITHANY-123/my-photography-sites`
2. Click **"Add file"** → **"Upload files"**
3. Drag and drop all your website files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `images` folder (if you have photos)
   - Any other files
4. Scroll down, type commit message: `Initial website deployment`
5. Click **"Commit changes"**
6. Wait for files to upload

### Step 2: Enable GitHub Pages

1. In your repository, click **"Settings"** (top menu)
2. Scroll down to **"Pages"** (left sidebar)
3. Under **"Source"**, select **"main"** branch
4. Click **"Save"**
5. Wait 1-2 minutes

### Step 3: Get Your Website URL

1. Refresh the Pages settings page
2. You'll see: **"Your site is live at"**
3. Your URL will be: `https://CHAITHANY-123.github.io/my-photography-sites`
4. **Copy this URL** - this is your live website!

## Method 2: Using Command Line (Git)

### Step 1: Open Command Prompt/Terminal

**Windows:**
- Press `Windows + R`
- Type `cmd` and press Enter

**Mac:**
- Open Terminal

### Step 2: Navigate to Your Website Folder

```bash
cd "E:\photoshop website"
```

### Step 3: Initialize Git (If Not Already Done)

```bash
git init
```

### Step 4: Add All Your Files

```bash
git add .
```

This adds all your files (index.html, styles.css, script.js, etc.)

### Step 5: Commit Your Files

```bash
git commit -m "Initial website deployment"
```

### Step 6: Connect to GitHub

```bash
git branch -M main
git remote add origin https://github.com/CHAITHANY-123/my-photography-sites.git
```

### Step 7: Push to GitHub

```bash
git push -u origin main
```

You'll be asked to enter your GitHub username and password (or token)

### Step 8: Enable GitHub Pages

1. Go to: `https://github.com/CHAITHANY-123/my-photography-sites`
2. Click **"Settings"** (top menu)
3. Click **"Pages"** (left sidebar)
4. Under **"Source"**, select **"main"** branch
5. Click **"Save"**
6. Wait 1-2 minutes

### Step 9: Get Your Website URL

Your website will be live at:
`https://CHAITHANY-123.github.io/my-photography-sites`

## Method 3: Using GitHub Desktop (Easiest GUI Method)

### Step 1: Download GitHub Desktop

1. Go to: **https://desktop.github.com/**
2. Download and install GitHub Desktop
3. Sign in with your GitHub account

### Step 2: Clone Your Repository

1. In GitHub Desktop, click **"File"** → **"Clone repository"**
2. Select **"my-photography-sites"**
3. Choose where to save it
4. Click **"Clone"**

### Step 3: Copy Your Website Files

1. Copy all your website files from `E:\photoshop website`
2. Paste them into the cloned repository folder

### Step 4: Commit and Push

1. In GitHub Desktop, you'll see all your files listed
2. Type commit message: `Initial website deployment`
3. Click **"Commit to main"**
4. Click **"Push origin"** (top right)
5. Wait for upload

### Step 5: Enable GitHub Pages

1. Go to: `https://github.com/CHAITHANY-123/my-photography-sites`
2. Click **"Settings"** → **"Pages"**
3. Select **"main"** branch
4. Click **"Save"**

### Step 6: Get Your URL

Your website: `https://CHAITHANY-123.github.io/my-photography-sites`

## Troubleshooting

### ❌ Files Not Uploading?

**Check:**
- ✅ All files selected?
- ✅ Internet connection working?
- ✅ GitHub account signed in?

### ❌ GitHub Pages Not Working?

**Check:**
- ✅ Files uploaded to repository?
- ✅ `index.html` in root folder?
- ✅ GitHub Pages enabled?
- ✅ Selected "main" branch?
- ✅ Wait 1-2 minutes for deployment

### ❌ Website Not Loading?

**Check:**
- ✅ Correct URL? `https://CHAITHANY-123.github.io/my-photography-sites`
- ✅ GitHub Pages enabled?
- ✅ Wait a few minutes for first deployment
- ✅ Clear browser cache (Ctrl+F5)

### ❌ Command Line Errors?

**Common Issues:**
- **"git is not recognized"** → Install Git: https://git-scm.com/
- **"Authentication failed"** → Use GitHub token instead of password
- **"Repository not found"** → Check repository name and URL
