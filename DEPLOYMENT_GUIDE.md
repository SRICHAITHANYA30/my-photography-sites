# 🚀 How to Deploy Your Website

## Easiest Method: Netlify (Recommended - 2 Minutes)

### Step 1: Prepare Your Files

Make sure you have all these files in your folder:
- ✅ `index.html`
- ✅ `styles.css`
- ✅ `script.js`
- ✅ `images` folder (if you have photos)
- ✅ Any other files

### Step 2: Go to Netlify

1. Open your browser
2. Go to: **https://www.netlify.com/**
3. Click **"Sign up"** (top right)
4. Sign up with:
   - Email, OR
   - GitHub account (recommended), OR
   - Google account

### Step 3: Deploy Your Website

**Method A: Drag and Drop (Easiest)**

1. After signing up, you'll see the Netlify dashboard
2. Look for **"Sites"** or **"Add new site"**
3. Click **"Add new site"** → **"Deploy manually"**
4. You'll see a box that says **"Drag and drop your site output folder here"**
5. **Drag your entire `photoshop website` folder** into that box
6. Wait for upload (usually 10-30 seconds)
7. **Done!** Your site is live!

**Method B: Using Netlify Drop**

1. Go to: **https://app.netlify.com/drop**
2. Drag your `photoshop website` folder
3. Wait for upload
4. Get your URL instantly!

### Step 4: Get Your Website URL

After deployment:
- You'll see a URL like: `https://random-name-12345.netlify.app`
- **Copy this URL**
- This is your live website!
- Share it with anyone!

### Step 5: Customize Your URL (Optional)

1. In Netlify dashboard, click on your site
2. Go to **"Site settings"**
3. Click **"Change site name"**
4. Enter a custom name (e.g., `srichaithanya-digitals`)
5. Your new URL: `https://srichaithanya-digitals.netlify.app`

### Step 6: Update Your Website

**To update your website:**
1. Make changes to your files
2. Go to Netlify dashboard
3. Drag and drop the updated folder again
4. Changes go live instantly!

## Method 2: GitHub Pages (Free)

### Step 1: Create GitHub Account

1. Go to: **https://github.com/**
2. Click **"Sign up"**
3. Create free account

### Step 2: Create New Repository

1. After signing in, click **"+"** (top right)
2. Click **"New repository"**
3. Repository name: `photography-website` (or any name)
4. Make it **Public**
5. **Don't** check "Initialize with README"
6. Click **"Create repository"**

### Step 3: Upload Your Files

**Option A: Using GitHub Website**

1. In your new repository, click **"uploading an existing file"**
2. Drag and drop all your files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `images` folder (if you have one)
   - Any other files
3. Scroll down, type commit message: `Initial commit`
4. Click **"Commit changes"**

**Option B: Using GitHub Desktop (Easier)**

1. Download GitHub Desktop: **https://desktop.github.com/**
2. Install and sign in
3. Click **"File"** → **"Add local repository"**
4. Select your `photoshop website` folder
5. Click **"Publish repository"**
6. Files upload automatically!

### Step 4: Enable GitHub Pages

1. In your repository, click **"Settings"** (top menu)
2. Scroll down to **"Pages"** (left sidebar)
3. Under **"Source"**, select **"main"** branch
4. Click **"Save"**
5. Wait 1-2 minutes

### Step 5: Get Your Website URL

1. Refresh the Pages settings page
2. You'll see: **"Your site is live at"**
3. URL will be: `https://YOUR_USERNAME.github.io/photography-website`
4. **Copy this URL** - this is your live website!

### Step 6: Update Your Website

**To update:**
1. Make changes to your files
2. Upload updated files to GitHub
3. Changes go live automatically (may take 1-2 minutes)

## Method 3: Vercel (Free)

### Step 1: Sign Up

1. Go to: **https://vercel.com/**
2. Click **"Sign up"**
3. Sign up with GitHub (recommended) or email

### Step 2: Deploy

1. Click **"Add New Project"**
2. If using GitHub, select your repository
3. Or drag and drop your folder
4. Click **"Deploy"**
5. Wait 1-2 minutes

### Step 3: Get Your URL

- You'll get a URL like: `https://your-project.vercel.app`
- Your site is live!

## Method 4: Traditional Web Hosting

### If You Have Web Hosting:

1. **Get FTP Details** from your hosting provider:
   - FTP Host: `ftp.yourdomain.com`
   - Username: `your-username`
   - Password: `your-password`

2. **Use FTP Client:**
   - Download FileZilla (free): **https://filezilla-project.org/**
   - Connect using your FTP details
   - Upload all files to `public_html` or `www` folder

3. **Your Website URL:**
   - Usually: `https://yourdomain.com`
   - Or: `https://www.yourdomain.com`

## Troubleshooting

### ❌ Website Not Loading?

**Check:**
- ✅ All files uploaded?
- ✅ `index.html` in root folder?
- ✅ Wait 1-2 minutes for deployment
- ✅ Clear browser cache (Ctrl+F5)

### ❌ Form Not Working?

**Check:**
- ✅ EmailJS configured correctly?
- ✅ Check browser console (F12) for errors
- ✅ Internet connection working?

### ❌ Images Not Showing?

**Check:**
- ✅ `images` folder uploaded?
- ✅ Image paths correct?
- ✅ File names match exactly (case-sensitive)
